Observations:
- The secure workspace was created with docs, src, and logs folders for project separation.
- Permissions were tightened to 750 for directories and 640 for files so only the owner can read or write them.
- The default umask value was 0022, which keeps newly created files mostly readable by others and directories accessible to the owner/group.
- These permissions protect project data by limiting access to the owner and preventing wider exposure.
